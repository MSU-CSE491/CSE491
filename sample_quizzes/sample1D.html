Multiple Choice Quiz

1. Given the following templated function:

  }

2. What is the output of the following program:

  star_map[0] += "*";

  star_map[0] += "*";
  std::cout << star_map[0] << "-" << star_map[1] << "-" << star_map[2];

3. What error is in the following function?
